# test/fixtures/pngsuite/reference

163 reference renderings (`.bgra` files) for the PngSuite conformance tests. Each file contains the expected raw BGRA pixel output for the corresponding PNG image.

Generated by decoding each valid PngSuite image with a known-correct decoder (PIL/Pillow) and converting to BGRA byte order. Tests compare the PNG decoder's output byte-for-byte against these references.
